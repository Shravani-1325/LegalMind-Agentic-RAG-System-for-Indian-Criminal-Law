# ⚖️ LegalMind: Agentic RAG for Criminal Law Intelligence

## Project Structure
The project follows a modular architecture with separate components for configuration, data ingestion, retrieval, and agent logic. Core functionalities are organized inside the `src/` directory, with configuration files in `config/` and supporting data stored in `data/`.

## Status
Work in progress — core RAG pipeline (ingestion, embeddings, and retrieval) is implemented, and agent workflow in `nodes.py` is currently under development.
